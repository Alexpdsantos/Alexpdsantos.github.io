OS System Sistema de Ordens de Serviço, Seus pedidos são uma ordem-
Este sistema foi desenvolvido utilizando a linguagem Java no
Back End, adotando todas as características e conceitos de
Orientação a objetos. A api vem sendo mantida na plataforma do Heroku.
No front end foi utilizado o framework Angular com a
linguagem TypeScript e foi publicado e está sendo mantido no GitHub Pages.
O Sistema de Ordens de serviço tem com objetivo atender
as demandas de ordens de serviço a serem executadas em uma determinada empresa,
de modo que o mesmo ajude aos Técnicos e clientes a ter
um controle melhor para cada ordem a ser executada.
